# @trixter-osec/core

[![npm](https://img.shields.io/npm/v/@trixter-osec/core.svg?color=blue)](https://www.npmjs.com/package/@trixter-osec/core)
[![Docs](https://img.shields.io/badge/docs-typedoc-blue)](https://coral-xyz.github.io/anchor/ts/index.html)

TypeScript client for Anchor programs.

## Note

- `@trixter-osec/core` depends on node.js native modules. Therefore, webpack 5 will not work with current version. You will either need to rollback to webpack 4, or use a polyfill for each missing dependency.
